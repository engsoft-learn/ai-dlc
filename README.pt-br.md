## 📖 O que é AI-DLC?

O **AI-DLC** (**AI-Driven Development Life Cycle**) é uma metodologia que direciona agentes de IA em todo o ciclo de vida de desenvolvimento de software (SDLC) — desde a intenção de negócio até a produção — mantendo o humano no controle. Ele funciona como um fluxo de trabalho estruturado com fases e artefatos, evitando as falhas do "vibe-coding" e trazendo a disciplina de engenharia para o desenvolvimento assistido por IA.
